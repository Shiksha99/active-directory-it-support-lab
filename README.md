🔹 Active Directory IT Support Home Lab

This project demonstrates the setup and administration of a Windows Active Directory environment in a virtual lab using Oracle VirtualBox.

📌 Overview
- Deployed a Windows Server as a Domain Controller (DC01)
- Created a domain environment (corp.local)
- Configured Organizational Units, users, and security groups
- Joined a Windows client machine (CLIENT01) to the domain
- Simulated IT support tasks including password resets and account management
- Configured shared folder access using group-based permissions
- Implemented basic Group Policy for centralized management

🧰 Technologies Used
- Windows Server
- Windows 10/11
- Active Directory Domain Services (AD DS)
- Oracle VirtualBox
- Group Policy Management
- DNS and Networking
  
🔧 Key Tasks Performed
- Installed and configured Active Directory Domain Services
- Promoted server to Domain Controller (corp.local)
- Created Organizational Units (HR, Sales, IT, Employees)
- Managed user accounts and security groups
- Joined client system to domain and verified authentication
- Performed password resets and user account disabling
- Configured shared network folder with group-based access
- Applied basic Group Policy settings

🎯 Skills Demonstrated
- Active Directory Administration
- IT Support & Help Desk Operations
- User and Access Management
- Network Configuration (DNS, IP)
- Troubleshooting & System Integration
